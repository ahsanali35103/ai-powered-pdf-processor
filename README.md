# AI Powered PDF Processor

An AI-powered document processing system built with **Laravel**, **Python**, and **OCR technology** to automate PDF processing, text extraction, and intelligent document workflows.

---

## 🚀 Overview

The AI Powered PDF Processor is designed to extract, process, and analyze PDF documents using Optical Character Recognition (OCR) and backend automation. The project combines Laravel for backend management with Python workers to perform intelligent document processing tasks.

---

## ✨ Features

* PDF Upload & Processing
* OCR Text Extraction
* AI Document Processing
* Backend Queue Processing
* RESTful APIs
* Authentication & Authorization
* Python Worker Integration
* Error Handling
* Database Management

## ⚙️ Installation

```bash
git clone https://github.com/ahsanali35103/ai-powered-pdf-processor.git

cd ai-powered-pdf-processor

composer install

cp .env.example .env

php artisan key:generate

php artisan serve
```
